# Guía de Contribución

¡Gracias por tu interés en contribuir a la Plataforma de Gestión de Eventos Comunitarios!

Sigue estas pautas para asegurar una colaboración clara, ordenada y productiva.

---

## 1. Requisitos previos
- Lee el README.md para entender el proyecto y su propósito.
- Instala las dependencias necesarias (ver sección de instalación en README).
- Familiarízate con la estructura del repositorio y las ramas principales.

## 2. Flujo de trabajo
1. **Busca una issue**: Elige una tarea en el tablero de issues o solicita una nueva.
2. **Crea una rama**: Usa el formato `feature/nombre-tarea`, `fix/nombre-bug` o `docs/nombre-doc`.
3. **Haz tus cambios**: Sigue las convenciones de código y añade pruebas si es necesario.
4. **Haz commit**: Mensajes claros y descriptivos. Ejemplo: `feat: agrega formulario de registro`.
5. **Haz push**: Sube tu rama al repositorio remoto.
6. **Crea un Pull Request (PR)**: Relaciona la issue correspondiente y describe los cambios.
7. **Revisión**: Espera feedback, realiza ajustes si es necesario y responde a los comentarios.
8. **Merge**: Solo el responsable del repositorio puede hacer merge tras la aprobación.

## 3. Convenciones de código
- Usa el estilo de código definido en el proyecto (Prettier, ESLint, etc.).
- Nombra variables y funciones de forma clara y descriptiva.
- Escribe comentarios solo cuando sea necesario para clarificar lógica compleja.
- Mantén el código modular y reutilizable.

## 4. Buenas prácticas para Pull Requests
- Relaciona tu PR con la issue correspondiente usando `Closes #número-issue`.
- Incluye capturas de pantalla o gifs si tu cambio afecta la UI.
- Asegúrate de que el PR pase todas las pruebas y linters.
- No mezcles cambios no relacionados en un mismo PR.

## 5. Comunicación
- Usa el canal de comunicación del equipo para dudas o bloqueos.
- Sé respetuoso y constructivo en los comentarios y revisiones.
- Si tienes sugerencias de mejora, crea una issue de tipo `enhancement`.

## 6. Código de conducta
- Este proyecto sigue un código de conducta inclusivo y respetuoso.
- No se tolerarán actitudes discriminatorias, ofensivas o irrespetuosas.
